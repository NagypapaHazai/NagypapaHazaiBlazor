Rövid leírás: A Nagypapa Házai egy ASP.NET Core Blazor alapú webalkalmazás, amely vidéki ingatlanok közösségi foglalását teszi lehetővé.
A felhasználók böngészhetik az elérhető házakat, foglalást hozhatnak létre megadott időszakra, és egy interaktív, színkódolt naptáron követhetik nyomon az összes foglalást.
A rendszer automatikusan megakadályozza az átfedő foglalásokat, és adminisztrátori felületet is biztosít a kezeléshez.
Tagok: Bota Csongor, Kania Áron Nontawat, Batsukh Temuujin
